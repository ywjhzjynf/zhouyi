## 一、系统简介

周易占卜系统是一个基于传统周易文化的占卜程序，支持多种起卦方式和完整的卦象解析。

### 核心功能

✅ **四种起卦方式:**
- 铜钱起卦（金钱卦）- 最传统的方法
- 蓍草起卦（大衍之数）- 最古老的方法
- 数字起卦 - 最便捷的方法
- 时间起卦（梅花易数）- 最灵活的方法

✅ **完整卦象解析:**
- 六十四卦卦辞
- 三百八十四爻爻辞
- 本卦、变卦、互卦
- 卦象象征意义

✅ **历史记录:**
- 自动保存占卜记录
- 支持查询历史卦象

## 二、快速开始

### 2.1 安装

**前提条件:** Python 3.10 或更高版本

```bash
# 1. 进入项目目录
cd zhouyi

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python main.py
```

### 2.2 第一次占卜

**推荐新手使用「铜钱起卦」:**

1. 运行程序后，选择 `1. 铜钱起卦`
2. 心诚默念所问之事（如：事业、感情、健康）
3. 点击回车模拟抛掷铜钱（共 6 次）
4. 查看卦象解析

## 三、起卦方法详解

### 3.1 铜钱起卦（推荐）

**适用场景:** 重要决策、人生大事

**操作步骤:**
1. 准备三枚铜钱（或硬币）
2. 心中默念所问之事
3. 抛掷六次，记录每次结果
4. 程序自动生成卦象

**铜钱正反面:**
- 正面（字）= 3
- 反面（花）= 2

**三枚总和:**
- 6 = 老阴（⚋ ×）- 变爻
- 7 = 少阳（⚊）- 不变
- 8 = 少阴（⚋）- 不变
- 9 = 老阳（⚊ ○）- 变爻

### 3.2 蓍草起卦

**适用场景:** 传统仪式、重要祭祀

**操作步骤:**
1. 选择「蓍草起卦」
2. 按照程序提示完成「四营十八变」
3. 每三变生成一爻（共六爻）

**传统流程:**
- 大衍之数五十，其用四十有九
- 分二、挂一、揲四、归奇
- 十八变成一卦

**注意:** 此方法较复杂，建议熟悉后再用。

### 3.3 数字起卦

**适用场景:** 快速占卜、日常小事

**操作步骤:**
1. 选择「数字起卦」
2. 输入任意数字（推荐 3-6 位数）
3. 程序自动转换为卦象

**示例:**
- 输入：`123456`
- 程序自动计算上下卦和动爻

### 3.4 时间起卦（梅花易数）

**适用场景:** 突发奇想、即时占卜

**操作步骤:**
1. 选择「时间起卦」
2. 使用当前时间或输入指定时间
3. 程序按梅花易数规则起卦

**计算规则:**
- 上卦 = 年 + 月 + 日 除以 8 取余
- 下卦 = 年 + 月 + 日 + 时 除以 8 取余
- 动爻 = 年 + 月 + 日 + 时 除以 6 取余

## 四、卦象解读

### 4.1 理解卦象

**卦象组成:**
- **本卦:** 初始卦象，代表当前状态
- **变卦:** 动爻变化后的卦象，代表发展趋势
- **互卦:** 由本卦中间四爻组成，代表过程

**爻位含义:**
- 初爻：事物开始
- 二爻：崭露头角
- 三爻：小成阶段
- 四爻：进入上层
- 五爻：成功巅峰
- 上爻：物极必反

### 4.2 读懂卦辞

**卦辞结构:**
- **卦名:** 如「乾为天」「坤为地」
- **卦辞:** 总体判断（如「元亨利贞」）
- **象曰:** 卦象象征（如「天行健，君子以自强不息」）

**常见术语:**
- 元：开始、根本
- 亨：通达、顺利
- 利：有利、适宜
- 贞：正固、坚持
- 吉：吉祥
- 凶：凶险
- 吝：困难
- 咎：过失
- 无咎：没有过失

### 4.3 理解爻辞

**爻辞格式:**
```
[爻位]: [爻辞内容]
```

**示例:**
```
初九：潜龙勿用
  - 初九：第一爻（阳爻）
  - 潜龙勿用：龙潜伏在水中，不宜行动
```

**爻位命名:**
- 阳爻：九（初九、九二、九三...）
- 阴爻：六（初六、六二、六三...）

## 五、常见问题

### Q1: 占卜准确吗？

**答:** 周易是传统文化遗产，提供的是哲学思考和决策参考，而非科学预测。建议理性看待，作为自我反思的工具。

### Q2: 一天可以占卜几次？

**答:** 传统讲究「初筮告，再三渎，渎则不告」。建议：
- 同一问题只占一次
- 重大事项可隔日再占
- 不可儿戏频繁占卜

### Q3: 什么是动爻？

**答:** 动爻是卦中发生变化的爻：
- 老阳（9）变阴
- 老阴（6）变阴
- 动爻代表变化的关键点

### Q4: 本卦和变卦怎么看？

**答:** 
- **本卦:** 当前状况
- **变卦:** 发展趋势
- 结合两者综合判断

### Q5: 如何保存卦象记录？

**答:** 程序自动保存到 `data/divination_history.db`，可查看历史记录功能（开发中）。

## 六、文化礼仪

### 6.1 占卜前的准备

**心态:**
- 心诚则灵
- 专心致志
- 不疑不占

**环境:**
- 安静整洁
- 避免干扰
- 可焚香净手（可选）

### 6.2 占卜禁忌

**不宜占卜的情况:**
- 酒后、情绪激动时
- 玩笑、戏谑心态
- 违法乱纪之事
- 伤害他人之事

**传统讲究:**
- 子时（23:00-1:00）不占
- 雷雨天气不占
- 身体不适不占

## 七、技术支持

### 7.1 遇到问题？

**常见问题排查:**

1. **程序无法启动**
   - 检查 Python 版本（需 3.10+）
   - 检查依赖是否安装：`pip install -r requirements.txt`

2. **显示乱码**
   - 确保终端支持 UTF-8
   - Windows 用户运行：`chcp 65001`

3. **数据错误**
   - 检查 `hexagrams.json` 是否完整
   - 运行 `python audit_data.py` 验证数据
