## 二、触摸操作说明

### 2.1 基本手势

| 手势 | 功能 |
|------|------|
| 单击 | 选择/确认操作 |
| 双击 | 查看详情（卦象界面） |
| 长按 | 复制文本/弹出菜单 |
| 滑动 | 切换页面/滚动内容 |
| 捏合 | 缩放卦象图（如有） |

### 2.2 起卦操作

#### 铜钱起卦

1. **点击「开始起卦」按钮**
2. **点击屏幕任意位置抛掷铜钱**
   - 每次点击模拟抛掷一次
   - 共需抛掷 6 次（生成 6 爻）
   - 每次抛掷有振动反馈
3. **等待自动生成卦象**

**操作提示：**
- 抛掷前建议静心想念所问之事
- 每次抛掷间隔建议 3-5 秒
- 可点击「重新起卦」重新开始

#### 蓍草起卦

1. **点击「开始起卦」按钮**
2. **跟随程序引导完成「四营十八变」**
   - 每变点击屏幕确认
   - 程序自动模拟蓍草分合
3. **十八变完成后生成卦象**

**注意：** 此过程较长（约 2-3 分钟），请保持耐心。

#### 数字起卦

1. **点击数字输入框**
2. **使用键盘输入数字**
   - 推荐 3-6 位数
   - 可输入任意正整数
3. **点击「确认」生成卦象**

#### 时间起卦

1. **点击「开始起卦」按钮**
2. **选择使用当前时间或指定时间**
   - 当前时间：直接点击确认
   - 指定时间：选择日期和时间
3. **点击「确认」生成卦象**

### 2.3 卦象界面操作

| 操作 | 功能 |
|------|------|
| 点击卦名 | 查看卦名详解 |
| 点击爻位 | 查看该爻爻辞详解 |
| 滑动左右 | 切换本卦/变卦/互卦 |
| 长按卦辞 | 复制文本 |
| 双击卦象 | 全屏查看 |

### 2.4 历史记录操作

|------|------|

| 长按记录 | 弹出菜单（分享/导出/删除） |
| 下拉列表 | 刷新记录 |

## 三、起卦方法详解

### 3.1 铜钱起卦（推荐新手）

**适用场景：** 日常占卜、重要决策

**操作步骤：**

```
1. 打开应用，点击「铜钱起卦」
        ↓
2. 静心想念所问之事（建议 1-2 分钟）
        ↓
3. 点击「开始起卦」
        ↓
4. 点击屏幕抛掷铜钱（共 6 次）
   - 第 1 次 → 初爻（最下方）
   - 第 2 次 → 二爻
   - 第 3 次 → 三爻
   - 第 4 次 → 四爻
   - 第 5 次 → 五爻
   - 第 6 次 → 上爻（最上方）
        ↓
5. 自动生成卦象和解读
```

**原理说明：**
- 正面（字）= 3，反面（花）= 2
- 三枚总和：6=老阴，7=少阳，8=少阴，9=老阳
- 老阴、老阳为变爻

**手机特色功能：**
- 📳 振动反馈：每次抛掷有轻微振动
- 🔊 音效：铜钱落地声（可在设置中关闭）
- 🎨 动画：铜钱翻转动画

---

### 3.2 蓍草起卦（传统古法）

**适用场景：** 重大决策、传统仪式

**操作步骤：**

```
1. 打开应用，点击「蓍草起卦」
        ↓
2. 静心想念所问之事
        ↓
3. 点击「开始起卦」
        ↓
4. 跟随引导完成「四营十八变」
   - 每三变生成一爻
   - 共十八变生成六爻
        ↓
5. 生成卦象和解读
```

**传统流程说明：**
- **大衍之数五十，其用四十有九**
- **四营：** 分二、挂一、揲四、归奇
- **十八变：** 三变成一爻，六爻共十八变

**手机特色功能：**
- 🎨 蓍草动画：可视化展示分合过程
- ⏱️ 进度提示：显示当前进行到第几变
- 📖 古法说明：每步有详细说明

---

### 3.3 数字起卦（快速便捷）

**适用场景：** 快速占卜、日常小事

**操作步骤：**

```
1. 打开应用，点击「数字起卦」
        ↓
2. 在输入框中输入数字
        ↓
3. 点击「确认」
        ↓
4. 生成卦象和解读
```

**数字选择建议：**

| 推荐 | 说明 |
|------|------|
| 随机想到的数字 | 最自然，最推荐 |
| 电话号码后几位 | 方便记忆 |
| 车牌号 | 个人相关 |
| 门牌号 | 住所相关 |

**不推荐：**
- ❌ 刻意选择的「吉利数字」
- ❌ 别人的隐私数字

**原理说明：**
- 上卦：数字除 8 取余
- 下卦：数字各位相加之和除 8 取余
- 动爻：数字除 6 取余

---

### 3.4 时间起卦（梅花易数）

**适用场景：** 特定时机、应景占卜

**操作步骤：**

```
1. 打开应用，点击「时间起卦」
        ↓
2. 选择时间来源
   - 当前时间：直接使用
   - 指定时间：手动选择
        ↓
3. 点击「确认」
        ↓
4. 生成卦象和解读
```

**原理说明（梅花易数）：**
- 上卦：（年 + 月 + 日）÷ 8 取余
- 下卦：（年 + 月 + 日 + 时）÷ 8 取余
- 动爻：（年 + 月 + 日 + 时）÷ 6 取余

**手机特色功能：**
- 🕐 精确到分钟：使用精确时间起卦
- 📅 历史时间：可选择过去或未来时间
- 🌍 时区支持：自动识别本地时区

## 四、卦象解读

### 4.1 卦象显示

```
╔════════════════════════════════════════════╗
║            【乾为天】                      ║
║            (䷀)                           ║
╠════════════════════════════════════════════╣
║                                            ║
║  6 ───────   上九：亢龙有悔               ║
║  5 ───────   九五：飞龙在天，利见大人     ║
║  4 ───────   九四：或跃在渊，无咎         ║
║  3 ───────   九三：君子终日乾乾...        ║
║  2 ───────   九二：见龙在田，利见大人     ║
║  1 ───────   初九：潜龙勿用               ║
║                                            ║
╠════════════════════════════════════════════╣
║ 卦辞：元亨利贞                             ║
║ 象曰：天行健，君子以自强不息               ║
╚════════════════════════════════════════════╝
```

### 4.2 符号说明

| 符号 | 含义 | 说明 |
|------|------|------|
| `───────` | 阳爻（少阳） | 不变 |
| `── ──` | 阴爻（少阴） | 不变 |
| `──⊙──` | 老阳 | 变爻（阳变阴） |
| `──✕──` | 老阴 | 变爻（阴变阳） |

### 4.3 本卦、变卦、互卦

**本卦：**
- 初始卦象
- 代表当前状态、事情的现状

**变卦：**
- 动爻变化后的卦象
- 代表发展趋势、事情的未来走向

**互卦：**
- 由本卦中间四爻组成
- 代表事情发展的过程

**查看方法：**
- 在卦象界面左右滑动切换
- 或点击底部标签切换

### 4.4 爻位含义

| 爻位 | 含义 | 象征 |
|------|------|------|
| 初爻 | 事物开始 | 萌芽阶段 |
| 二爻 | 崭露头角 | 初步发展 |
| 三爻 | 小成阶段 | 局部成功 |
| 四爻 | 进入上层 | 接近核心 |
| 五爻 | 成功巅峰 | 最佳状态 |
| 上爻 | 物极必反 | 盛极而衰 |

### 4.5 卦辞术语解释

| 术语 | 含义 |
|------|------|
| 元 | 开始、根本、大 |
| 亨 | 通达、顺利 |
| 利 | 有利、适宜 |
| 贞 | 正固、坚持 |
| 吉 | 吉祥 |
| 凶 | 凶险（提醒谨慎） |
| 吝 | 困难、艰难 |
| 咎 | 过失 |
| 无咎 | 没有过失 |

## 五、历史记录

### 5.1 查看历史记录

1. 点击主界面底部「历史记录」
2. 列表显示最近的占卜记录
3. 点击任意记录查看详情

### 5.2 记录内容

每条记录包含：
- 📅 占卜日期和时间
- 🪙 起卦方法
- ☯️ 本卦卦象
- 🔀 变卦卦象（如有）
- 📝 占问事项（如填写）
- 📌 备注

### 5.3 记录管理

**搜索记录：**
- 点击顶部搜索框
- 输入卦名、日期、关键词

**筛选记录：**
- 按起卦方法筛选
- 按日期范围筛选

**删除记录：**
- 左滑单条记录 → 删除
- 长按 → 批量删除

**导出记录：**
- 长按记录 → 导出
- 可导出为图片或文本
- 可分享到微信、QQ 等
