## 第六步：解读卦象

### 关键信息

1. **卦名**：如"乾为天"、"坤为地"等
2. **卦辞**：总体吉凶判断
3. **象曰**：形象化的解释
4. **动爻**：变化的关键位置

### 快速判断

| 情况 | 解读 |
|------|------|
| 无动爻 | 事情稳定，按当前趋势发展 |
| 1-2 个动爻 | 有变化，关注动爻位置 |
| 3 个以上动爻 | 变化较大，需谨慎 |
| 全变 | 彻底转变，重新考虑 |

## 进阶技巧

### 1. 选择合适的时间

- **最佳**：清晨、心静时
- **避免**：情绪激动、酒后、深夜

### 2. 提问的艺术

✅ **好的问题：**
- "这个项目是否值得投入？"
- "我应该如何改善人际关系？"
- "最近的健康状况如何？"

❌ **避免的问题：**
- "我会发财吗？"（太宽泛）
- "他/她喜欢我吗？"（涉及他人隐私）
- 同一问题反复占问

### 3. 理解变卦

```
本卦 → 当前状态
  ↓
变卦 → 发展趋势
```

- 本卦好，变卦差：先吉后凶，需谨慎
- 本卦差，变卦好：先难后易，坚持
- 本卦变卦都好：大吉，可行动
- 本卦变卦都差：暂缓，重新考虑

## 常见问题速查

**Q: 结果不好怎么办？**
A: 卦象是提醒，不是判决。不好的卦象提示你需要注意的方面，可以调整策略。

**Q: 可以帮别人占吗？**
A: 可以，但最好由当事人亲自占问，心诚则灵。

**Q: 多久可以占一次？**
A: 同一问题建议只占一次。不同问题可以随时占。
