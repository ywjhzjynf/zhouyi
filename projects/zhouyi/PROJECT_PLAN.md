**数据来源要求：**
- 以《周易正义》（唐·孔颖达）为权威底本
- 参考《周易折中》（清·康熙）校勘
- 所有数据需标注来源出处

**数据格式标准（与兵部、工部确认）：**
```json
{
  "hexagrams": [
    {
      "id": 1,
      "name": "乾",
      "pinyin": "qián",
      "symbol": "䷀",
      "binary": "111111",
      "judgment": "元亨利贞",
      "image": "天行健，君子以自强不息",
      "lines": [
        {
          "position": 1,
          "text": "潜龙勿用",
          "type": "yang"
        }
      ]
    }
  ]
}
```

### 应急预案

| 触发条件 | 应急措施 |
