### 第二部分：核心功能演示（5 分钟）

#### 2.1 铜钱起卦（推荐演示）

**演讲词:**

> 首先，我演示最常用的「铜钱起卦」方法。
> 
> 传统上，我们需要三枚铜钱，抛掷六次。现在，我们用程序模拟这个过程。

**操作:**
```bash
python main.py
# 选择：1. 铜钱起卦
```

**互动环节:**
> 在开始之前，请大家心中默想一个问题，比如：
> - 最近的工作发展如何？
> - 是否应该换工作？
> - 新的投资机会怎么样？

**模拟抛掷（6 次）:**
```
第 1 次抛掷：7（少阳）⚊
第 2 次抛掷：8（少阴）⚋
第 3 次抛掷：9（老阳）⚊ ○  ← 动爻
第 4 次抛掷：7（少阳）⚊
第 5 次抛掷：8（少阴）⚋
第 6 次抛掷：7（少阳）⚊
```

**解读卦象:**
```
════════════════════════════════════════
本卦：火天大有 (䷍)
卦辞：元亨
象曰：火在天上，大有。君子以遏恶扬善，顺天休命

初九：无交害，匪咎，艰则无咎
九二：大车以载，有攸往，无咎
九三：公用亨于天子，小人弗克
九四：匪其彭，无咎
六五：厥孚交如，威如，吉  ← 动爻
上九：自天祐之，吉无不利

变卦：乾为天 (䷀)
════════════════════════════════════════
```

**解读要点:**
- **本卦「大有」:** 象征丰收、富有
- **卦辞「元亨」:** 非常顺利
- **动爻（六五）:** 以诚信待人，有威严，吉祥
- **变卦「乾」:** 最终会达到刚健成功的状态

**结论:** 这是一个很好的卦象，表示只要保持诚信和威严，就能获得成功。

**演讲词:**

**操作:**
```bash

# 选择：4. 时间起卦
# 选择：1. 使用当前时间
```

**示例输出:**
```
起卦时间：2026-03-20 14:30
上卦：坎（水）
下卦：离（火）
动爻：第 4 爻

本卦：水火未济 (䷿)
变卦：山水蒙 (䷃)
```

**解读:**
> 「未济」表示事情尚未完成，还需要努力。
> 「蒙」表示启蒙、学习，暗示需要向他人请教。
