# 周易占卜系统 v1.0.0 - 手机版使用说明

**应用名称：** 周易占卜  
**版本：** v1.0.0  
**适用平台：** Android 5.0+  
**文件大小：** 约 30MB

---

## 📥 安装方法

### 方法一：直接安装 APK

1. 下载 `zhouyi-1.0.0-debug.apk` 文件
2. 在手机上打开文件管理器
3. 点击 APK 文件
4. 如果提示"未知来源"，请前往设置允许安装
5. 点击"安装"
6. 安装完成后点击"打开"

### 方法二：通过 ADB 安装

```bash
adb install zhouyi-1.0.0-debug.apk
```

### 一、铜钱起卦（金钱卦）

**适用场景：** 传统起卦，心诚则灵

**操作步骤：**

1. 点击"🪙 铜钱起卦"按钮
2. 心中默念所问之事
3. 点击"🎲 抛掷铜钱"按钮
4. 重复 6 次，每次生成一爻
5. 自动显示卦象和卦名

**示例：**

```
第初爻：7 - 少阳（不变）
第二爻：8 - 少阴（不变）
第三爻：9 - 老阳（变爻）
第四爻：7 - 少阳（不变）
第五爻：8 - 少阴（不变）
上爻：6 - 老阴（变爻）

【水火既济】
```

**说明：**
- 6 = 老阴（变爻）
- 7 = 少阳（不变）
- 8 = 少阴（不变）
- 9 = 老阳（变爻）

---

### 二、蓍草起卦（大衍之数）

**适用场景：** 传统演算，仪式感强

**操作步骤：**

1. 点击"🌿 蓍草起卦"按钮
2. 心中默念所问之事
3. 点击"开始演算"按钮
4. 系统自动完成"十八变"
5. 显示卦象和卦名

**说明：**
- 大衍之数五十，其用四十有九
- 四营成易：分二、挂一、揲四、归奇
- 三变成一爻，十八变成一卦

---

### 三、数字起卦

**适用场景：** 快速起卦，随机应变

**操作步骤：**

1. 点击"🔢 数字起卦"按钮
2. 点击"点击输入数字"按钮
3. 输入任意数字（如生日、幸运数字）
4. 点击"起卦"按钮
5. 显示卦象和卦名

**示例：**

```
输入数字：12345

第初爻：8
第二爻：7
第三爻：8
第四爻：7
第五爻：9
上爻：8

【火天大有】
```

---

### 四、时间起卦（梅花易数）

**适用场景：** 即时起卦，应时而动

**操作步骤：**

1. 点击"⏰ 时间起卦"按钮
2. 查看当前时间
3. 点击"以当前时间起卦"按钮
4. 系统根据年月日时自动起卦
5. 显示卦象和卦名

**原理：**
- 上卦：（年 + 月 + 日）÷ 8 取余
- 下卦：（年 + 月 + 日 + 时）÷ 8 取余
- 动爻：（年 + 月 + 日 + 时）÷ 6 取余

## ❓ 常见问题

### 1. 应用闪退

**原因：** 手机系统版本过低  
**解决：** 确保 Android 版本 ≥ 5.0

### 2. 显示乱码

**原因：** 字体不支持中文  
**解决：** 更新系统或更换手机

### 3. 触摸不灵敏

**原因：** 屏幕保护膜影响  
**解决：** 检查屏幕或重启应用

### 4. 如何重新起卦

**方法：** 点击"返回主菜单"，重新选择起卦方法
