## 一、安装与配置

### Q1.1: 如何安装 Python？

**答:** 

**Windows 用户:**
```powershell
# 方法 1: 使用 winget（推荐）
winget install Python.Python.3.11

# 方法 2: 从官网下载
# 访问 https://www.python.org/downloads/
# 下载安装 Python 3.11 或更高版本
```

**macOS 用户:**
```bash
# 使用 Homebrew
brew install python@3.11
```

**Linux 用户:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3.11 python3-pip

# CentOS/RHEL
sudo yum install python3.11 python3-pip
```

**验证安装:**
```bash
python --version  # 应显示 Python 3.11.x
```

### Q1.2: 依赖安装失败怎么办？

**答:**

1. **升级 pip:**
```bash
python -m pip install --upgrade pip
```

2. **逐个安装依赖:**
```bash
pip install rich
pip install click
```

3. **使用国内镜像:**
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q1.3: 程序运行时提示编码错误？

**答:**

**Windows 用户:**
```powershell
# 设置控制台编码为 UTF-8
chcp 65001

# 然后运行程序
python main.py
```

**永久解决方案:**
在系统环境变量中添加：
```
PYTHONIOENCODING=utf-8
```

## 二、使用方法

### Q2.1: 哪种起卦方法最准确？

**答:** 

四种方法各有特点：

| 方法 | 准确度 | 难度 | 适用场景 |
|------|--------|------|----------|
| 铜钱起卦 | ⭐⭐⭐⭐ | 简单 | 重要决策 |
| 蓍草起卦 | ⭐⭐⭐⭐⭐ | 复杂 | 传统仪式 |
| 数字起卦 | ⭐⭐⭐ | 简单 | 日常小事 |
| 时间起卦 | ⭐⭐⭐⭐ | 中等 | 即时占卜 |

**推荐:** 新手从「铜钱起卦」开始，熟悉后可尝试其他方法。

### Q2.2: 铜钱起卦时，用硬币可以吗？

**答:** 可以。传统用三枚铜钱，现代可用硬币代替：
- 1 元硬币即可
- 正面（有数字面）= 3
- 反面（有花面）= 2

**注意:** 最好固定使用同一套硬币，保持连续性。

### Q2.3: 数字起卦应该输入什么数字？

**答:** 

**推荐:**
- 随机想到的数字（3-6 位数）
- 电话号码后几位
- 车牌号
- 门牌号

**不推荐:**
- 刻意选择的「吉利数字」
- 别人的隐私数字（如身份证号）

**原则:** 心诚则灵，数字本身不重要，重要的是起卦时的心念。

### Q2.4: 时间起卦用哪个时区？

**答:** 程序默认使用系统本地时间。如需指定时区，可在代码中修改（高级用户）。

### Q2.5: 如何查看历史占卜记录？

**答:** 

当前版本功能开发中，预计 v0.2.0 上线。

**临时方案:** 手动记录到笔记本，包含：
- 占卜日期时间
- 所问之事
- 得到的卦象
- 事后验证

## 三、卦象解读

### Q3.1: 看不懂卦辞怎么办？

**答:**

**建议步骤:**

1. **先读白话翻译**
   - 卦辞下方通常有现代汉语解释
   - 关注关键词：吉、凶、亨、利

2. **理解卦象象征**
   - 如「乾为天」象征刚健、进取
   - 「坤为地」象征柔顺、包容

3. **结合自身情况**
   - 卦辞是普遍原理
   - 需结合具体问题灵活理解

4. **参考多个来源**
   - 不要仅依赖单一解释
   - 可查阅《周易译注》等书籍

### Q3.2: 什么是「变卦」？

**答:**

**变卦**是由动爻变化产生的新卦：

**示例:**
```
本卦：乾为天 (䷀)
  - 九三爻动（老阳变阴）
变卦：天泽履 (䷉)
```

**解读方法:**
- **本卦:** 当前状况
- **变卦:** 发展趋势
- **动爻:** 变化的关键点

### Q3.3: 多个动爻怎么看？

**答:**

**传统规则:**
- **一个动爻:** 以该爻爻辞为主
- **两个动爻:** 以上爻爻辞为主，参考下爻
- **三个动爻:** 以本卦和变卦卦辞综合判断
- **四个动爻:** 以变卦卦辞为主
- **五个动爻:** 以变卦不变爻爻辞为主
- **六个动爻:** 
  - 乾、坤卦用「用九」「用六」
  - 其他卦以变卦卦辞为主

### Q3.4: 「无动爻」是什么意思？

**答:**

**无动爻**表示所有爻都是少阳（7）或少阴（8），没有老阳（9）或老阴（6）。

**解读:**
- 以本卦卦辞为主
- 事态稳定，变化不大
- 宜守不宜攻

### Q3.5: 卦象显示「凶」怎么办？

**答:**

**正确理解:**

1. **「凶」不是宿命**
   - 是提醒，不是判决
   - 提示潜在风险

2. **趋吉避凶**
   - 卦辞说「凶」，是让你谨慎
   - 调整行为可改善

3. **参考建议**
   - 如「利涉大川」表示适宜行动
   - 如「不利有攸往」表示宜静不宜动

**心态:** 周易是决策参考，不是命运预言。

## 五、文化礼仪

### Q5.1: 占卜前需要斋戒吗？

**答:**

**传统要求:**
- 斋戒三日（不饮酒、不食肉）
- 沐浴更衣
- 焚香祷告

**现代简化:**
- 洗手净面
- 保持环境整洁
- 心态虔诚

**原则:** 心诚则灵，形式可简化。

### Q5.2: 什么时间不宜占卜？

**答:**

**传统禁忌:**
- 子时（23:00-1:00）
- 雷雨天气
- 日食、月食
- 自身或家人丧期

**现代建议:**
- 避免深夜（精神不佳）
- 避免酒后（神志不清）
- 避免情绪激动时

### Q5.3: 同一问题可以反复占吗？

**答:**

**传统原则:** 「初筮告，再三渎，渎则不告」

**建议:**
- 同一问题只占一次
- 如结果不明，可隔日再占
- 不可因结果不好而反复占

### Q5.4: 可以替别人占卜吗？

**答:**

**可以，但需注意:**
- 需征得对方同意
- 不可用于窥探隐私
- 不可用于伤害他人

**最佳方式:** 让当事人自己占卜。

## 六、技术支持

### Q6.1: 程序报错「ModuleNotFoundError」？

**答:**

**原因:** 缺少依赖包

**解决:**
```bash
# 重新安装依赖
pip install -r requirements.txt

# 或手动安装
pip install rich click
```

### Q6.2: 程序报错「FileNotFoundError」？

**答:**

**原因:** 数据文件缺失

**解决:**
1. 检查 `zhouyi/data/hexagrams.json` 是否存在
2. 如缺失，从备份恢复或重新下载
3. 运行验证：`python audit_data.py`

### Q6.3: 如何更新到最新版本？

**答:**

**当前版本:** v0.1.0

**更新方式:**
```bash
# 进入项目目录
cd zhouyi

# 拉取最新代码（如使用 git）
git pull origin main

# 重新安装依赖
pip install -r requirements.txt
```
