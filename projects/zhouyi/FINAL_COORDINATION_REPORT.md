## 🚀 使用说明

### 立即运行（Web 版）

```bash
# 1. 安装依赖
pip install flet

# 2. 运行应用
cd C:\Users\25243\.openclaw\workspace\projects\zhouyi
python zhouyi/mobile/main.py

# 3. 在浏览器中访问 http://localhost:8550
```

### 打包为 Windows 应用

```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 打包
pyinstaller --onefile --windowed --name="周易占卜" zhouyi/mobile/main.py

# 3. 可执行文件位于 dist/周易占卜.exe
```
