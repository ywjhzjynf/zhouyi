## 技术细节

### 八卦二进制标准
```
乾 ☰ = 111 (三阳)
兑 ☱ = 110 (下缺)
离 ☲ = 101 (中虚)
震 ☳ = 100 (仰盂)
巽 ☴ = 011 (下断)
坎 ☵ = 010 (中满)
艮 ☶ = 001 (覆碗)
坤 ☷ = 000 (三阴)
```

### 六十四卦二进制计算
```
六十四卦二进制 = 上卦二进制 + 下卦二进制
示例：既济卦 (上坎下离) = 010 + 101 = 010101
```
